# Venv Setup

- Run Command: "python -m venv venv"
- Activate: "./venv/Scripts/activate"
- Then Complete Pip commands below

# Python (Pip) Modules Dependency List

- pip install flask
- pip install pymongo
- pip install pyjwt==1.7.1
- pip install bcrypt
